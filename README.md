# h6-Miniprojekti
## Lähtövaatimus
Halutaan nopeasti, uusittavasti ja helposti konfiguroida useampi palvelin. Tehdään tätä varten ansiblella automaatiota, jolla saadaan ladattavien pakettien uusimmat versioit kohdekoneille ja käynnistettyä ne.
## Lopputulema
Saadaan parit webbipalvelimet ladattua, konffattua ja potkittua käyntiin ansible playbookilla.

## Jatkojalostus
Wordpress tuotantopalvelin, php, mariaDB, ufw palomuuriasetukset.
